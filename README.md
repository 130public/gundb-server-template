# GunDB with Node.js

This a production ready node.js app, tailored for Vercel serverless functions.

## Instructions

1. Setup environment variables, you can use the sample.env file

2. Install dependencies:

```sh
npm install
# or
yarn
  ```

3. Execute the following command:

  ```sh
yarn start
  ```
